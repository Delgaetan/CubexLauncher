# CubexLauncher

Un launcher personnalisé pour Minecraft développé en Python. Ce projet permet de lancer le jeu facilement en crack

## 🚀 Fonctionnalités

* Sélection de la version du jeu.
* Gestion des pseudos.
* Interface graphique simple et intuitive.

## 📋 Prérequis

Avant de lancer le projet, assure-toi d'avoir installé :

* [Python 3.x](https://www.python.org/)
* [Java](https://www.java.com/) (requis pour exécuter Minecraft)
